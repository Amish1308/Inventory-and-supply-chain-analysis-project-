# Inventory-and-supply-chain-analysis-project-
||PROJECT OVERVIEW||
This project focuses on analyzing inventory and supply chain data to evaluate operational performance and identify inefficiencies. The goal is to derive actionable insights for improving inventory management, reducing costs, and optimizing logistics.

||OBJECTIVES||
Analyze inventory and sales data
Track key supply chain KPIs
Identify inefficiencies in logistics and stock management
Build an interactive dashboard for decision-making

||TOOLS & TECHNOLOGIES||
Power BI
Data Cleaning & Transformation
DAX (for KPI calculations)

||KEY KPIs||
Inventory Turnover: 22.86
Days Sales of Inventory (DSI): 16.12 days
Warehouse Utilization: 34.94%
Backorder Analysis

||KEY INSIGHTS||
High backorders indicate supply-demand mismatch
Certain regions have significantly higher transportation costs
Inventory turnover suggests efficient stock movement but scope for optimization
Warehouse utilization is relatively low, indicating underused capacity

||BUSINESS IMPACT||
This dashboard helps:
Improve inventory planning
Reduce stockouts and overstocking
Optimize transportation costs
Support data-driven decision-making
