# Inventory & Supply Chain Analytics Dashboard

## Project Overview
This project analyzes inventory and supply chain data to identify inefficiencies and improve decision-making.

## Objectives
- Analyze inventory and sales data  
- Track supply chain KPIs  
- Identify logistics inefficiencies  
- Build an interactive dashboard  

## Tools & Technologies
- Power BI    
- DAX  

## Key KPIs
- Inventory Turnover: 22.86  
- Days Sales of Inventory (DSI): 16.12 days  
- Warehouse Utilization: 34.94%  

## Key Insights
- High backorders indicate supply-demand mismatch  
- Certain regions have higher transportation costs  
- Low warehouse utilization shows unused capacity  

## Dataset
- Product details  
- Sales data  
- Logistics data  
- Regional data  

## Business Impact
- Improved inventory planning  
- Reduced stockouts  
- Better cost optimization
